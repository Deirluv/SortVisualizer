# Sort Visualizer

An interactive desktop app built with **Avalonia** and **C#**. This project demonstrates how sorting algorithms work by showing step-by-step iterations that the program does to sort the list

---

## 🚀 Features

- **Real-time Visualization:** Watch how different algorithms manipulate data structures step-by-step.
- **Dynamic Speed Control** Adjustable delay between iterations
- **Customizable Data:** Choose the number of elements you want to generate
- **Sound effects:** Every element has a unique sound linked to its height (value in the list)

---

## 🛠 Tech Stack

* **Language:** C# (.NET 9)
* **Framework:** [Avalonia](https://avaloniaui.net/) - self-realized RelayCommands (both sync and async)
* **Pattern:** MVVM (Model-View-ViewModel).
* **Sound processing:** [NAudio](https://github.com/naudio/naudio)
